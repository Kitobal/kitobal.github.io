Gate Quest


Acerca de

Un videojuego para aprender sobre el concepto de las compuertas lógicas.
Versión 0.4
Incluye los 3 primeros niveles del juego.
(v0.4.1 arregla un error)

Objetivo

Obtener una salida 1 (verdadero) para pasar al siguiente nivel.

Controles

Teclado:
WASD o Flechas para moverse.
C o V para seleccionar la compuerta.
Barra espaciadora para interactuar.
Control de videojuego:
Palanca izquierda para moverse.
X o Y para seleccionar la compuerta.
A para interactuar.
