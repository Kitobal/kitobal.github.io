Delivery Guy


Objective

Deliver as many packages as you can before the time runs out.
Each time you deliver a package you gain 5 seconds.
Packages always start in the same buildings (the bigger ones).
You have to find the house where each package must be delivered.

Controls

Move: Arrow Keys or WASD Keys or Left stick on a controller.
(Only tested with an Xbox One Controller)
